# Safeher - Women's Safety App

**Safeher** is a comprehensive women safety app that aims to empower women by providing essential safety features such as Safe Routes, Crime Locations, and Emergency Alerts. With real-time notifications, voice recognition, and various self-defense tutorials, it serves as a digital companion for safety in any situation.

## 🔮 Features

### Core Safety Features
- 📍 **Safe Route:** Display safe routes on the map to help users navigate through secure pathways.
- 📍 **Safe Locations:** Mark and display verified safe locations on the map.
- 🚨 **Crime Locations:** View recent crime data on the map to stay alert and avoid risky areas.
- 📰 **Recent News:** Stay updated with the latest news and reports related to women's safety.
- 🥋 **Self-Defense Tutorials:** Access helpful self-defense tutorials for different situations.
- 📝 **Complaint Registration:** Easily file complaints with detailed descriptions for immediate action.
- 📝 **Anonymous Complaint Registration:** File complaints anonymously for safety and privacy.
- 🚨 **SoS Alerts:** Send emergency alerts with live location to all contacts with a single tap.
- 📞 **Fake Call Generator:** Escape awkward situations by generating fake calls to your phone.
- 🗣️ **Voice Recognition:** Detect keywords like "help," "save," "emergency," and send live location alerts to emergency contacts.
- 👮 **Admin Portal:** Manage and update the status of complaints in the admin portal.

### 🆕 New Advanced Features

#### 🤖 AI-Powered Safety Assessment
- **Real-time Safety Scoring:** Get dynamic safety scores (0-100) based on location, time, and crime data
- **Risk Factor Analysis:** Comprehensive assessment of time risk, location risk, population risk, and crime risk
- **Personalized Recommendations:** AI-generated safety tips based on current conditions and risk level
- **Safe Route Optimization:** Find the safest routes with real-time safety scoring

#### 📍 Advanced Location Tracking & Geofencing
- **Real-time Location Sharing:** Automatic location updates every 30 seconds during tracking
- **Smart Geofencing:** Create virtual boundaries around important locations (home, work, school)
- **Entry/Exit Alerts:** Get notified when entering or leaving geofenced areas
- **Missed Update Detection:** Automatic alerts if location updates stop unexpectedly
- **Zone-based Safety:** Different alert levels for safe zones, danger zones, and waypoints

#### 📱 Panic Shake Detection
- **Motion-Activated SOS:** Shake phone 3 times quickly to send emergency alerts
- **Adjustable Sensitivity:** Customizable shake detection sensitivity
- **Background Monitoring:** Works even when app is in background
- **Audio Feedback:** Immediate audio confirmation when SOS is triggered

#### 👥 Community Safety Network
- **Crowdsourced Safety Alerts:** Share and view safety alerts from nearby community members
- **Anonymous Reporting:** Post safety concerns anonymously while helping others
- **Safety Patrol Program:** Join community safety patrols to help keep neighborhoods safe
- **Real-time Community Feed:** See nearby safety incidents and helpful responses
- **Interactive Support:** Like and comment on safety alerts to provide community support

#### 🚨 Smart Emergency Response System
- **Intelligent Escalation:** Automatic escalation based on severity and time
- **Multi-level Response:** 4-level escalation from contacts to authorities to emergency services
- **Real-time Coordination:** Live tracking of emergency response actions
- **Emergency History:** Complete log of all emergency incidents and responses
- **Manual Override:** Ability to escalate or de-escalate emergencies as needed

#### ✈️ Travel Safety Companion
- **Destination Safety Analysis:** Comprehensive safety assessment for travel destinations
- **Journey Planning:** Create detailed travel plans with safety considerations
- **Check-in System:** Automated check-ins during travel with missed check-in alerts
- **Travel-specific Tips:** Contextual safety advice for different destinations
- **Emergency Travel Support:** Travel-specific emergency response coordination

## ⚙️ Installation

### Method 1: Manual Installation

1. **Fork this repository:** Click the Fork button located in the top-right corner of this page.
2. **Clone the repository:**
   ```bash
   git clone https://github.com/<your-username>/women-safety-app.git
   ```
3. **Set .env file:**
   Inside the frontend and backend directories set `.env`:

   Frontend:

   ```bash
   VITE_BACKEND_URL=<your_server_url>
   ```

   Backend:

   ```bash
   PORT=3000
   ```

4. **Install dependencies:**
   ```bash
   npm install     # Run in both client and server directories
   ```
5. **Start the servers:**  
   Frontend:
   ```bash
   cd client
   npm run dev
   ```
   Backend:
   ```bash
   cd server
   npm run dev
   ```
6. **Access the application:**
   ```bash
   http://localhost:5173/
   ```
